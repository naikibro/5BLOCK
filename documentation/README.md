# 5BLOCK Documentation

Documentation simple pour lancer la dApp localement.

## Lancer la documentation

```bash
# Depuis la racine du projet
pnpm run docs

# Ou depuis le dossier documentation
cd documentation
pnpm start
```

Ouvrir http://localhost:3001

## Build

```bash
cd documentation
pnpm build
```
